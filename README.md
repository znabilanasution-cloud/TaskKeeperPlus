# 📋 TaskKeeper Plus

TaskKeeper Plus adalah aplikasi **To-Do List** berbasis **React Native (Expo)** yang menggunakan **AsyncStorage** untuk menyimpan data secara lokal sehingga tugas tetap tersimpan meskipun aplikasi ditutup.

## ✨ Fitur
- ➕ Tambah, edit, hapus, dan tandai tugas selesai (CRUD)
- 💾 Penyimpanan data menggunakan AsyncStorage
- 🌙 Dark Mode
- 📊 Statistik tugas
- 🔍 Pencarian tugas
- 🗂️ Filter berdasarkan kategori
- 🔄 Sorting (Terbaru, A-Z, Status)
- 🧹 Hapus semua tugas

## 🛠️ Teknologi
- React Native
- Expo
- JavaScript
- AsyncStorage
- Context API

## 🚀 Menjalankan Project

```bash
npm install
npx expo start
```
